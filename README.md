# DisplayDimmer

DisplayDimmer is a simple Windows 11 utility that lets you dim selected monitors while leaving another monitor bright for watching movies, videos, or media at night.

## Features

- Select multiple displays to dim
- Leave one or more displays undimmed
- Click-through dimming overlays
- Emergency Esc shutoff
- Saved monitor and opacity settings
- Simple dark UI
